Das Flüster-Spiel

Wecke die kleinen Monster auf! Sie lieben die Stille. Wenn es im Raum zu laut wird, erschrecken sie und platzen weg. Schafft ihr es, dass alle Monster am Leben bleiben?

Hinweis: Das Spiel benötigt Zugriff auf dein Mikrofon, um die Lautstärke zu erfassen. Alle Daten bleiben lokal auf deinem Gerät.

Geräte-Kalibrierung

Kalibriere die Monster auf dein Laptop-Mikrofon:

  1. Öffne dieses Spiel auf einem **zweiten Gerät** (z.B. Handy).
  2. Stelle dieses Handy über das Menü oben rechts auf "Sender" um.
  3. Platziere das Handy ca. **1 Meter entfernt** von diesem Laptop.
  4. Starte den Testton auf dem Handy. Stelle sicher, dass der Raum ansonsten **still** ist.
  5. Klicke unten auf "Messen & Kalibrieren". Die Software misst den Ton für 3 Sekunden und gleicht sich automatisch ab.
